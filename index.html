hexo+github搭建个人博客

发表于 2018-07-24 | 分类于 学习笔记

准备工作

  • github账号
  • 安装node.js
  • 安装git for windows

    搭建github博客

    新建github名为.github.io的repo，username为你的github用户名
    配置ssh key
    ssh-keygen -t rsa -C “邮件地址”
    cd ~/. ssh
    将id_rsa.pub的内容贴到新建的ssh keys

    测试ssh -T git@github.com
    Hi xxx! You’ve successfully authenticated, but GitHub does not provide shell access.
    则表示成功

    安装hexo

    打开git bash 输入npm install -g hexo
    新建一个博客目录，在目录内右键打开git bash或者cd到该目录，初始化hexo init

hexo g # 生成

访问http://localhost:4000/即可看到内容

修改主题

clone主题到themes目录
修改_config.yml中的theme: landscape改为theme: yilia，然后重新执行hexo g来重新生成。

部署到github

首先安装npm install hexo-deployer-git –save，否则提交会报错。
修改_config.yml

打开git bash，输入hexo d -g #生成并上传。成功后即可通过https://username.github.io/访问
